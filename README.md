This is the start of my Linux and GIT command Cheat Sheet!
Protip: I highly recommend playing Terminus to practice Linux commands! 

Command #1 - Linux:
ls | This command is used in Linux and will give you a list of all files and directories in your current file location.

Command #2 - Linux:
cd | This command stands for 'change directory' and lets you switch between directories.

Command #3 - Linux:
mkdir | This command is used to create directories.

Command #4 - Linux:
touch | This command is used to create files.

Command #5 - Linux:
rm | This command is used to remove files.

Command #6 - Linux:
nano | This command opens up an alternate text editor that's very different than vi!

Command #7 - Linux:
pwd | This command stands for 'print working directory' and prints what directory you currently find yourself in.
